## Cómo ejecutar los ejercicios individuales

Desde esta carpeta, en la terminal:

```bash
python ejercicio-01.py
```

Reemplazar `01` por el número del ejercicio que se quiera ejecutar (`01` a `10`).

## Entrega final

La carpeta [`entrega/`](./entrega/) contiene la versión consolidada para subir al aula virtual:

| Archivo | Descripción |
|---------|-------------|
| [`entrega/tp1-estructuras-secuenciales.py`](./entrega/tp1-estructuras-secuenciales.py) | Único archivo `.py` con los 10 ejercicios y un menú interactivo |
| [`entrega/tp1-estructuras-secuenciales.zip`](./entrega/tp1-estructuras-secuenciales.zip) | Archivo `.zip` listo para subir al aula virtual |

Para ejecutar la versión consolidada:

```bash
cd entrega
python tp1-estructuras-secuenciales.py
```

El programa muestra un menú del 1 al 10 para elegir qué ejercicio ejecutar (0 para salir).
