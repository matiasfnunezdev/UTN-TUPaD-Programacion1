## Cómo ejecutar

1. Tener instalado **Python 3.10 o superior**.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:

```bash
python tp3-integrador-estructuras-repetitivas.py
```

4. Aparece un menú principal donde se puede elegir cualquiera de los 5 ejercicios:

```
============================================================
TP INTEGRADOR - REPETITIVAS, CONDICIONALES Y SECUENCIALES
============================================================
  1. Caja del Kiosco
  2. Acceso al Campus y Menú Seguro
  3. Agenda de Turnos (sin listas)
  4. Escape Room: La Bóveda
  5. Arena del Gladiador
  0. Salir
============================================================
```

## Ejercicios incluidos

| # | Título | Concepto principal |
|---|--------|--------------------|
| 1 | Caja del Kiosco | `for` + validaciones con `.isalpha()` y `.isdigit()` |
| 2 | Acceso al Campus | login con intentos + menú con `while` |
| 3 | Agenda de Turnos | menú repetitivo SIN listas/diccionarios |
| 4 | Escape Room: La Bóveda | mecánica de juego con anti-spam y bucles |
| 5 | Arena del Gladiador | combate por turnos con `for` y tipado int/float/str/bool |

## Restricciones cumplidas

- ❌ Sin `try`/`except`.
- ✅ Validaciones con `.isalpha()` y `.isdigit()` dentro de `while`.
- ✅ Uso de estructuras secuenciales, condicionales y repetitivas.
- ✅ Ejercicio 3: sin listas, diccionarios, sets ni tuplas.
- ✅ Ejercicio 5: uso explícito de `int`, `float`, `str` y `bool`.
