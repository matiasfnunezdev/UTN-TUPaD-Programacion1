**Tecnicatura Universitaria en Programación a Distancia (UTN)**
Materia: **Programación 1** - Unidad 5: Listas

```
tp5-integrador-listas/
├── tp5-integrador-listas.py        # Resolución consolidada con menú interactivo

└── README.md                       # Este archivo
```

## Requisitos

- **Python 3.8** o superior.
- No requiere bibliotecas externas (sólo `random` de la librería estándar).

## Ejecución

Desde una terminal, ubicado en la carpeta del proyecto:

```bash
python tp5-integrador-listas.py
```

Aparecerá un **menú interactivo** con los 13 ejercicios. Ingresá el número del ejercicio que querés ejecutar y seguí las indicaciones por consola. Para salir, opción `0`.

## Ejercicios incluidos

| #  | Ejercicio                                       |
|----|-------------------------------------------------|
| 1  | Notas de 10 estudiantes                         |
| 2  | 5 productos (ordenar y eliminar)                |
| 3  | 15 aleatorios entre 1 y 100: pares e impares    |
| 4  | Eliminar duplicados de una lista                |
| 5  | 8 estudiantes (agregar o eliminar)              |
| 6  | Rotación a la derecha                           |
| 7  | Temperaturas semanales (matriz 7x2)             |
| 8  | Notas 5 estudiantes x 3 materias                |
| 9  | Tablero de Ta-Te-Ti (3x3)                       |
| 10 | Ventas de 4 productos x 7 días (matriz 4x7)     |
| 11 | Búsqueda de nombre en una lista                 |
| 12 | Ordenar 8 enteros (asc/desc)                    |
| 13 | Ranking de puntajes                             |

## Notas

- Todas las visualizaciones de listas se realizan con **estructuras repetitivas** (`for`/`while`).
- Las validaciones de entrada se hacen mediante métodos de cadenas (`.isdigit()`, etc.) sin `try/except`.
- Los ejercicios usan únicamente recursos vistos en las unidades 1 a 5 (variables, condicionales, bucles, listas y listas anidadas).
