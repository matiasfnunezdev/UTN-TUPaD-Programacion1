# TP Integrador 5 - Listas

**Materia:** Programación 1
**Carrera:** Tecnicatura Universitaria en Programación a Distancia (UTN)
**Unidad:** 5 - Listas

## Estructura

```
tp5-integrador-listas/
├── tp5-integrador-listas-enunciado.md   # Enunciado completo
├── ejercicio-01.py … ejercicio-13.py    # Una solución por archivo
├── entrega/                             # Paquete final para subir al campus
│   ├── tp5-integrador-listas.py         # Consolidado con menú
│   ├── tp5-integrador-listas-enunciado.md
│   ├── README.md
│   └── tp5-integrador-listas.zip        # Archivo a subir
└── README.md
```

## Ejercicios

| #  | Archivo               | Descripción                                       |
|----|-----------------------|---------------------------------------------------|
| 1  | `ejercicio-01.py`     | Notas de 10 estudiantes: promedio, máx y mín      |
| 2  | `ejercicio-02.py`     | 5 productos (sorted + remove)                     |
| 3  | `ejercicio-03.py`     | 15 enteros aleatorios: pares e impares            |
| 4  | `ejercicio-04.py`     | Eliminar duplicados                               |
| 5  | `ejercicio-05.py`     | 8 estudiantes (agregar/eliminar)                  |
| 6  | `ejercicio-06.py`     | Rotación a la derecha                             |
| 7  | `ejercicio-07.py`     | Temperaturas semanales (matriz 7x2)               |
| 8  | `ejercicio-08.py`     | Notas 5 estudiantes x 3 materias                  |
| 9  | `ejercicio-09.py`     | Tablero de Ta-Te-Ti (3x3)                         |
| 10 | `ejercicio-10.py`     | Ventas 4 productos x 7 días (matriz 4x7)          |
| 11 | `ejercicio-11.py`     | Buscar nombre en una lista                        |
| 12 | `ejercicio-12.py`     | Ordenar 8 enteros (asc/desc)                      |
| 13 | `ejercicio-13.py`     | Ranking de puntajes                               |

## Cómo correr

Ejercicios individuales:

```bash
python ejercicio-XX.py
```

Versión consolidada con menú:

```bash
python entrega/tp5-integrador-listas.py
```

## Entrega

El archivo a subir al campus es:

`entrega/tp5-integrador-listas.zip`

Contiene:
- `tp5-integrador-listas.py` (consolidado con menú)
- `tp5-integrador-listas-enunciado.md`
- `README.md`
